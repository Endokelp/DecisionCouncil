# Contributing

Thanks for helping improve DecisionCouncil.

## Quick fixes

README clarity, typos, install path issues: PR welcome.

## Skill changes

Edit `skills/council/SKILL.md`. Keep:

- `disable-model-invocation: true`
- Lite mode as default
- Scout/Fact Checker only for last30days and deep-research
- Token caps on sub-agents

Test in Cursor (`council this:`) or Claude Code (`/council`) before submitting.
